# Doppler Speed Estimator using Verilog

## Overview
This project estimates the speed of a moving object using Doppler frequency shift.

## Features
- Frequency estimation
- Direction detection
- Speed calculation
- Verilog RTL implementation

## Tools Used
- Verilog
- Vivado
- Digital Signal Processing
- EDA Playground
